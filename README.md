# ai_voice-agent
# 🎙️ AI Voice Agent (Python)

An interactive AI-powered voice assistant built with Python that listens through your microphone, understands speech using AI, and speaks back with a natural, delightful voice using OpenAI’s streaming Text‑to‑Speech.

---

## ✨ Features

- 🎤 Live microphone speech input
- 🧠 Conversational AI responses with memory
- 🔊 Streaming Text-to-Speech playback
- ♾️ Continuous conversation loop
- ⚡ Low-latency async audio playback
- 🧩 Clean, extensible architecture

---

## 🧱 Tech Stack

- **Python 3.9+**
- **SpeechRecognition** – Speech-to-Text
- **OpenAI Responses API** – LLM inference
- **OpenAI Streaming TTS** – Voice output
- **sounddevice + numpy** – PCM audio playback
- **asyncio** – Async audio streaming
